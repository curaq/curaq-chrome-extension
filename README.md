# CuraQ Saver - Chrome Extension

[CuraQ](https://curaq.pages.dev) に記事を保存するための公式Chrome拡張機能です。

**CuraQ** は、保存した記事をAIが自動整理し、あなたに最適な記事を毎日配信する知識キュレーションサービスです。

この拡張機能を使うことで、Web上の記事を簡単に保存できます。

## 特徴

- **誰でも使える**: 無料プランでも利用可能
- **プライバシー優先**: URLのみ送信。ページ内容は送信されません
- **簡単操作**: 右クリックメニューまたは拡張アイコンクリックで即座に保存
- **Proプランでさらに便利に**: ポップアップ内で確認して即座に保存

## 必要なもの

- [CuraQ](https://curaq.pages.dev) のアカウント（無料）
- Google Chrome ブラウザ

## インストール方法

### Chrome Web Store（準備中）

*現在、Chrome Web Storeへの公開準備中です。*

### 手動インストール

1. **拡張機能をダウンロード**
   - [こちら](https://github.com/togishima/curaq-chrome-extension) のページで、緑色の「Code」ボタンをクリック
   - 「Download ZIP」を選択してダウンロード
   - ダウンロードしたZIPファイルを展開

2. **Chromeに読み込み**
   - Chromeで `chrome://extensions/` を開く
   - 右上の「デベロッパーモード」をONにする
   - 「パッケージ化されていない拡張機能を読み込む」をクリック
   - 展開したフォルダを選択

3. **準備完了**
   - [CuraQ](https://curaq.pages.dev/login) にログインしていることを確認
   - これで拡張機能が使えるようになります

## 使い方

### 基本の使い方

1. 保存したい記事を開く
2. 以下のいずれかの方法で保存：
   - **右クリックメニュー**: ページ上で右クリック → 「CuraQに保存」を選択
   - **拡張アイコン**: ブラウザツールバーの拡張アイコンをクリック → 「この記事を保存」

### プランによる違い

**無料・Standardプラン**
- CuraQの確認画面が新しいタブで開きます
- 確認画面で「保存する」をクリックして完了

**Proプラン（APIトークン設定時）**
- ポップアップ内で保存内容を確認
- URLとタイトルのみ送信（ページ内容は送信されません）
- 確認後、即座に保存完了

**APIトークンの設定方法**
1. 拡張アイコンをクリック
2. 設定アイコン（歯車マーク）をクリック
3. [CuraQ設定ページ](https://curaq.pages.dev/settings/access-token)でトークンを取得
4. トークンを入力して保存

**プライバシーとセキュリティ**
- 送信されるのは**URLとタイトルのみ**です
- ページの本文や機微情報は送信されません
- サーバー側でURLから記事内容を取得するため、プライバシーリスクが最小限です

## トラブルシューティング

### 保存できない場合

1. **CuraQにログインしているか確認**
   - [CuraQ](https://curaq.pages.dev/login) にログインしてください
   - ログインしていない場合、確認画面が表示されます

2. **それでも解決しない場合**
   - [フィードバックページ](https://curaq.pages.dev/feedback)からお問い合わせください

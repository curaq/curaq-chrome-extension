## CuraQとは？

**CuraQ** は、保存した記事をAIが自動で整理し、あなたに最適な記事を毎日お届けする知識キュレーションサービスです。

この拡張機能を使うと、Webで見つけた記事をワンクリックでCuraQに保存できます。

## この拡張機能でできること

- 右クリックメニューから記事を保存
- 拡張機能アイコンをクリックして記事を保存
- 保存前に内容を確認できる

## 必要なもの

- **CuraQアカウント**（無料で作成できます）: [CuraQ](https://curaq.app)
- **Google Chrome** ブラウザ

## インストール方法

Chrome Web Storeから簡単にインストールできます。

### ステップ1: Chrome Web Storeからインストール

1. [Chrome Web Store - CuraQ Saver](https://chrome.google.com/webstore) にアクセスします
   > 🔗 **リンクは公開後に更新されます**
2. **「Chromeに追加」** ボタンをクリックします
3. 確認ダイアログで **「拡張機能を追加」** をクリックします

これでインストール完了です！

### ステップ2: 拡張機能をツールバーに固定する（推奨）

インストール後、ツールバーに固定しておくと便利です。

1. Chromeの右上にあるパズルピースのアイコン（拡張機能メニュー）をクリックします
2. 「CuraQ Saver」の横にあるピンアイコンをクリックします
3. ツールバーにCuraQのアイコンが常に表示されるようになります

## 使い方

### 方法1: 右クリックメニューから保存

1. 保存したい記事のページを開きます
2. ページ上で **右クリック** します
3. メニューから **「CuraQに保存」** を選択します
4. 確認画面が表示されるので、内容を確認して **「送信」** をクリックします

### 方法2: 拡張機能アイコンから保存

1. 保存したい記事のページを開きます
2. ツールバーの **CuraQアイコン** をクリックします
3. **「この記事を保存」** をクリックします
4. 確認画面で内容を確認して **「送信」** をクリックします

## プライバシーについて

この拡張機能が送信するのは **URLとページタイトルのみ** です。

- ページの本文は送信されません
- ログイン情報やパスワードは送信されません
- CuraQのサーバーがURLから記事内容を取得します

## うまく動かないときは

### 「保存できない」場合

1. **CuraQにログインしていますか？**
   - [CuraQ](https://curaq.app) を開いてログイン状態を確認してください

2. **拡張機能が有効になっていますか？**
   - `chrome://extensions/` を開いて、「CuraQ Saver」がONになっているか確認してください

3. **ページを再読み込みしてみてください**
   - 拡張機能をインストールした直後は、ページの再読み込みが必要な場合があります

### それでも解決しない場合

[フィードバックページ](https://curaq.app/feedback)から状況をお知らせください。
できる限り対応いたします。

## アップデート方法

Chrome Web Storeからインストールした場合、拡張機能は**自動的に更新**されます。

新しいバージョンがリリースされると、Chromeが自動的にアップデートを適用します。特別な操作は必要ありません。

手動で更新を確認したい場合は、`chrome://extensions/` を開き、画面左上の「更新」ボタン（丸い矢印）をクリックしてください。
